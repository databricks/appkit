## Documentation

The `docs/` directory contains the AppKit documentation site, built with Docusaurus.

**Working with docs:**

```bash
# From root
pnpm docs:dev    # Start dev server
pnpm docs:build  # Build docs
pnpm docs:serve  # Serve built docs
```
